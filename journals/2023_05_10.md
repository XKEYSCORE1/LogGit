- The Spread `(...)` Operator
	- The `...` operator expands an iterable (like an array) into more elements
		- ``` javascript
		  const q1 = ["Jan","Feb","Mar"];
		  const q2 = ["Apr","May","Jun"];
		  const q3 = ["Jul","Aug","Sep"];
		  const q4 = ["Oct","Nov","Dec"];
		  
		  const year = [...q1, ...q2, ...q3, ...q4];
		  
		  
		  ```
